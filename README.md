# CyberPlay 🎮

**CyberPlay** is a PHP-based game hub that features a user login system and a stylish GUI, giving users access to a collection of fun online games — all in one place!

## 🚀 Features

- 🔐 **Login System** – Secure and simple user authentication using PHP sessions.
- 🎨 **Cool GUI** – A modern, eye-catching interface for an engaging user experience.
- 🕹️ **Game Links** – Direct access to a variety of browser-based games.
- 📱 **Responsive Design** – Works smoothly on both desktop and mobile browsers.

## 📁 Technologies Used

- PHP
- HTML & CSS
- JavaScript (for interactivity)
- PHP Sessions for login handling
